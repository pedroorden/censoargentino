# censoargentino — Dónde encontrar el paquete

`censoargentino` está disponible como paquete Python en PyPI y como proyecto de código abierto en GitHub.

---

## Instalación

```bash
pip install censoargentino
```

---

## PyPI

[pypi.org/project/censoargentino](https://pypi.org/project/censoargentino/)

El paquete se distribuye a través del Python Package Index (PyPI). Cualquier entorno Python con `pip` puede instalarlo directamente — sin clonar repositorios ni configuración adicional. Las actualizaciones se publican con `pip install --upgrade censoargentino`.

---

## GitHub

[github.com/pedroorden/censoargentino](https://github.com/pedroorden/censoargentino)

El código fuente es público y está disponible en GitHub bajo licencia MIT. Ahí se pueden ver los commits, reportar problemas y seguir el desarrollo del proyecto.

---

## Dataset (Hugging Face)

[huggingface.co/datasets/pedroorden/censoargentino](https://huggingface.co/datasets/pedroorden/censoargentino)

Los archivos Parquet que usa el paquete están alojados en Hugging Face Datasets. `censoargentino` los consulta directamente vía DuckDB — no es necesario descargarlos manualmente.
